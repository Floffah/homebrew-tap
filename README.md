# Floffah Tap

## How do I install these formulae?

`brew install floffah/tap/<formula>`

Or `brew tap floffah/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "floffah/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
